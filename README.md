# Fav Speedy - E-commerce Fashion Store

A modern, responsive e-commerce platform for fashion products including bags, shoes, and clothing for both men and women.

## Features

- **Product Catalog**: Browse through extensive collections of men's and women's fashion items
- **Shopping Cart**: Add items to cart with real-time updates and cart count display
- **Checkout System**: Multi-step checkout process with shipping and payment information
- **Payment Options**: Multiple payment methods including:
  - Credit/Debit Cards
  - Bank Transfer
  - Paystack
  - Flutterwave
  - Cash on Delivery
- **Order Tracking**: Track your orders after purchase
- **Responsive Design**: Mobile-friendly interface that works on all devices

## Product Categories

### Women's Section
- Bags (16+ premium designs)
- Shoes (16+ styles)
- Clothes

### Men's Section
- Bags (14+ professional designs)
- Shoes
- Clothes

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap 5
- LocalStorage for cart management

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Browse products and add items to cart
4. Proceed to checkout to complete your purchase
